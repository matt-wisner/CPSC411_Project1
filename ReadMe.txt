To Run:
1. Extract Claim_RestServer and Project2 to a folder
2. In Claim_RestServer Folder(IntelliJ application):
	Change path "dbName" in Database.kt to point to claimsDB.db
3. In Project2 folder (Android studio application):
	Change path "requestURL" in ClaimService.kt to proper IP address
4. Run Claim_RestServer 
5. Run Project2
6. Type information to fill in claim and click add button
7. Check status message to see if claim was successfully added
